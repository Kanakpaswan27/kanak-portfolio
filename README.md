# 🚀 Modern React Developer Portfolio

> **A premium, responsive, and interactive developer portfolio built with React.js, Tailwind CSS, Framer Motion, and Vite.**

This portfolio represents my journey as a **BCA 2nd Year student** and aspiring frontend developer. It is designed to showcase my projects, technical skills, and development experience through a clean, modern, and responsive user interface.

The portfolio features smooth animations, a cinematic hero section, optimized performance, responsive layouts, and an elegant user experience across desktop, tablet, and mobile devices.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Kanakpaswan27/kanak-portfolio.git
```

Navigate to project

```bash
cd kanak-portfolio
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build project

```bash
npm run build
```
